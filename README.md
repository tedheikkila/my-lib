  ## Description

  myLib is a PC and mobile app designed to function as a placeholder.
  
  ## Technologies:

  * Front-end: React/Jsx, HTML, CSS, Js
  * Back-end: mongoose, MongoDB, Robo 3T, express, apollo server, GraphQL
  * Dev-deps: nodemon, Chrome's dev tools
